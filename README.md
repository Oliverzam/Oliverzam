## 🚀 Proyectos destacados

### 🏢 [Sistema de Control de Asistencia](https://github.com/Oliverzam/attendance-system)
Sistema multiplataforma con API REST (+70 endpoints), panel admin en Angular y app móvil en Flutter. Geolocalización, geofencing, notificaciones push via Firebase FCM y despliegue con Docker.

`Node.js` `Express` `Angular` `Flutter` `PostgreSQL` `Docker` `Firebase FCM`

### 🌿 [Sistema de Detección de Malezas con IA](https://github.com/Oliverzam/weed-detection-system)
Tesis de grado. Detecta y cuantifica malezas en cultivos de papa desde imágenes de drones usando Mask R-CNN. +90% de precisión en validación. Expuesto como microservicio REST.

`Python` `TensorFlow` `Keras` `Mask R-CNN` `Flask` `Docker`
